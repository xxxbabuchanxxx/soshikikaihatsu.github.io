# SSAP提案システム デプロイチェックリスト

システムを本番環境にデプロイする前に、以下の項目を確認してください。

## ✅ 必須設定

### 1. Google Apps Script

- [ ] Googleスプレッドシートを作成済み
- [ ] スプレッドシートIDを`backend/gas_backend.gs`の`SPREADSHEET_ID`に設定
- [ ] GASをWebアプリとしてデプロイ済み
- [ ] デプロイ設定で「アクセスできるユーザー: 全員」に設定
- [ ] `setupTriggers`関数を実行してトリガーを設定済み
- [ ] GASのデプロイURLを取得済み

### 2. フロントエンド設定

- [ ] `js/config.js`の`GAS_WEB_APP_URL`にGASデプロイURLを設定
- [ ] `TOTAL_EMPLOYEES`を正しい従業員数に設定（`js/config.js`と`backend/gas_backend.gs`の両方）
- [ ] `PROPOSAL_DURATION_DAYS`を適切な期間に設定（必要に応じて）

### 3. GitHub設定

- [ ] GitHubリポジトリを作成済み
- [ ] リポジトリの可視性を設定（Public/Private）
- [ ] GitHub Pagesを有効化
- [ ] GitHub Pagesのブランチとフォルダを設定（通常は`main`ブランチ）

## 🚀 デプロイ手順

### GitHub Pagesへのデプロイ

```bash
# 1. リポジトリの初期化
cd ssap_proposal_system
git init

# 2. 全てのファイルをステージング
git add .

# 3. 初回コミット
git commit -m "Initial commit: SSAP提案システム"

# 4. メインブランチに変更
git branch -M main

# 5. リモートリポジトリを追加
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git

# 6. プッシュ
git push -u origin main
```

### GitHub Pagesの有効化

1. GitHubのリポジトリページで「Settings」をクリック
2. 左メニューから「Pages」を選択
3. Source: `main`ブランチを選択
4. Folder: `/ (root)`を選択
5. 「Save」をクリック
6. 数分待つと公開URLが表示されます
